  • Orders

  • How do I create an order?

    Tap on the Orders menu in the bottom navigation bar.
    On the Orders page, Tap on the Create Order option.
    On the create order screen, give a name to your order, tap on Create Order.
    On the Order Details page, look up for the product using the search bar provided and add the product to an order.
    Alternatively, tap on the scan icon available next to the search box and scan the barcode of the product you wish to order.

  • What is upload order?

    Once you have added all the items to your cart, you may choose to upload this order to OE web for further analysis and review.
    You can tap on the Upload order option to see this order on OE Web for further analysis. On successful upload, the order is no longer available on the app

  • What is place order?

    Once you have added all the items to your cart, you can directly submit this order for processing using this option on the app. On successful submission, the order is removed from the cart.
    Note - If your order contains controlled items, we would like you to know that the direct submission of this order is not possible on the app.

  • How do I delete an order?

    On the Cart page, you will see all the active orders, swipe left on the order tile to see the delete option, tap on the delete option to delete the order.
